[디바이스 언어 대응](https://kakaoent.atlassian.net/browse/KS-10433)

- 디바이스 언어: 유저가 설정하는 시스템 언어 (앱 메뉴 등)
	- ko, th, en
- 콘텐츠언어: 유저가 설정 가능한 앱 옵션. 서비스 내 노출하는 콘텐츠 제어
	- ko, th

- API에는 아래 값을 헤더로 전달(+앱에서 웹뷰에 헤더로 2가지 정보를 내려준다)
```
X-Device-Lang // 디바이스 언어 
X-Service-Lang // 서비스 언어 (콘텐츠 언어 / 주사용 언어)
```

## 수정 방향

- 기존 로케일은 제거한다. path도 제거한다
- 디바이스 언어와 서비스 언어 관리
	- 앱
		- 디바이스 언어: 장치 언어를 따라간다?
		- 서비스 언어: 페이지에서 설정
	- 웹뷰
		- 앱에서 헤더로 디바이스언어, 서비스언어 전달해준다
	- 웹
		- 디바이스 언어: Accept-language 를 활용하여 관리
		- 서비스언어
			- 로그인 유저: app-init이나 어떤 API를 통해 값을 가져온다.
			- 미로그인 유저: 디바이스 언어를 따라간다.
