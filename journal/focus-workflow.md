# 🔁 Daily Focus Workflow

> [[task-dashboard|📋 대시보드]] | [[journal/1-w/|📅 주간 리뷰]]

## 규칙 2개
1. 할 일은 아무 노트에나 `- [ ] 할 일`로 적기 (마감 있으면 `📅 YYYY-MM-DD`)
2. 1시간 집중 후보는 태그 추가: `#focus/study` / `#focus/dev` / `#focus/stock`

## 아침 3분
- 대시보드 열기 → "오늘/연체"와 "#focus/*"에서 **오늘의 1시간 딱 1개** 고르기
- 데일리 노트에 적기

## 실행 (25분 × 2)
- 25분 → 체크 또는 "다음 행동 1줄" → 휴식 → 25분 → 체크
- 부담되면 **15분만** 해도 성공

## 저녁 2분
- 데일리 종료 섹션 채우기 (한 줄 기록 + 내일 1개 + 오늘 노트 회고)
- 연체 태스크: 처리 / 내일로 이월(`📅`) / 마감 제거(백로그로)
