# 🔁 Daily Focus Workflow (매일 1시간)

목표: **투두는 어디에 적어도 OK** / **매일은 한 화면에서 결정** / **1시간 집중은 절차로 실행**

## 평소(수시로) — 캡처 규칙
- 떠오르는 할 일은 아무 노트에나 `- [ ] 할 일`로 기록
- 마감이 있으면 `📅 YYYY-MM-DD` 추가 (예: `📅 2026-02-24`)
- 1시간 집중 후보면 태그 1개 추가(선택)
  - `#focus/study` / `#focus/dev` / `#focus/stock`

## 매일 아침(3~5분) — 오늘 확정
1) `/journal/task-dashboard.md` 열기
2) `오늘/연체`에서 **오늘 처리할 것 0~2개만** 고르기
3) `오늘만`에서 **오늘 처리할 것 0~2개만** 고르기
4) `#focus/*`에서 **오늘의 1시간(딱 1개)** 고르기
5) 데일리 노트(`journal/0-d/YYYY-MM-DD.md`) 맨 위에 오늘의 1시간을 1줄로 박기
   - 예: `- [ ] 오늘의 1시간: 백로그에서 고른 것 (25m + 25m)`

## 실행(55~60분) — 25분 × 2
- 25분 #1 → 끝나면 체크(또는 “다음 행동 1줄”만 추가)
- 5~10분 휴식
- 25분 #2 → 끝나면 체크

## 저녁(2분) — 정리/이월
- 대시보드에서 `오늘/연체`가 남아있으면 **처리/이월/마감 제거** 중 하나로 정리
- 내일 할 거면: `📅 내일`로 바꾸기
- “언젠가”면: 마감일을 제거해서 `마감일 없는 백로그`로 보내기

## 주 1회(10분) — 백로그 다듬기
- `마감일 없는 백로그`에서 “너무 큰 일”을 **25분짜리 다음 행동**으로 쪼개기
- 진짜 안 할 거면 체크 해제/삭제/아카이브(본인 방식)

## 막힐 때(최소 규칙)
- 오늘의 1시간이 부담되면: **15분만** 시작하고 “다음 행동 1줄”만 남기면 성공 처리

## Troubleshooting
- 대시보드가 비어 보이면: `Settings → Tasks → Global Filter` 확인
- ` ```tasks ` 블록이 그냥 코드처럼 보이면: Tasks 플러그인 설치/활성화 확인
