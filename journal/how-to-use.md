# 📖 옵시디언 사용 가이드

> 이 시스템의 핵심: **적는 건 아무 데서나, 관리는 대시보드 한 곳에서, 돌아보기는 주간 리뷰에서**

---

## 매일 반복

### 아침 (3분) — "오늘 뭐 하지?"
1. [[task-dashboard|📋 대시보드]] 열기
2. "오늘/연체"에서 급한 거 확인
3. "1시간 집중 후보"에서 **딱 1개** 고르기
4. 데일리 노트 생성 (Templater: `Alt+N` 또는 캘린더 클릭)
5. "오늘의 1시간"에 고른 거 적기

### 낮 — 할 일이 떠오를 때
- 지금 보고 있는 노트에 바로 `- [ ] 할 일` 적기
- 마감이 있으면 `📅 2026-03-15` 붙이기
- 1시간 집중감이면 `#focus/dev` 등 태그 추가
- **어디에 적든 대시보드가 자동으로 모아줌** → 정리할 필요 없음

### 낮 — 아이디어/메모가 떠오를 때
- 데일리 노트의 "떠오른 생각"에 한 줄 적기
- 길어지면 `0-inbox/inbox.md`에 던져두기 (주간 리뷰에서 정리)

### 낮 — 1시간 집중 실행
- 25분 타이머 → 작업 → 체크 또는 "다음 행동" 메모
- 5분 쉬고 → 25분 한 번 더
- 부담되면 **15분만 해도 OK** (시작이 중요)

### 저녁 (2분) — "오늘 뭐 했지?"
1. 데일리 노트 종료 섹션 채우기:
   - **한 줄 기록**: 오늘 뭐 했는지 한 문장
   - **내일의 1개**: 내일 아침에 바로 시작할 거
   - **오늘 노트 회고**: 만들거나 수정한 노트 이름 적기
2. 대시보드에서 연체 태스크 처리:
   - 할 거면 → `📅 내일날짜`로 변경
   - 안 할 거면 → 마감일 삭제 (백로그로 이동됨)
   - 끝난 거면 → 체크

---

## 매주 반복

### 일요일 (10분) — 주간 리뷰
1. `journal/1-w/`에 주간 리뷰 노트 생성 (Templater로 `weekly-review` 템플릿 사용)
2. 순서대로 채우기:
   - **한 줄 회고**: 이번 주를 한 문장으로
   - **최근 7일 노트**: Dataview가 자동으로 보여줌 → 훑어보고 다시 볼 거 메모
   - **투두 정리**: 백로그에서 안 할 거 삭제, 큰 거 쪼개기
   - **다음 주 집중**: 다음 주에 꼭 하고 싶은 거 1개

## 케이스별 가이드

### "할 일이 너무 많아서 뭐부터 해야 할지 모르겠다"
→ 대시보드만 보기. "오늘/연체"가 진짜 급한 거. 나머지는 무시해도 됨.

### "할 일을 적어놓고 까먹는다"
→ 마감일(`📅`)만 붙이면 대시보드 "오늘/연체"에 자동으로 뜸. 까먹을 수 없음.

### "데일리 노트를 안 쓰게 된다"
→ 아침에 대시보드 여는 것만 습관 들이기. 거기서 데일리 노트 링크 누르면 자동 생성됨.
→ 종료 섹션은 3줄뿐이라 부담 없음. 안 채워도 괜찮고, 채우면 보너스.

### "노트를 쓰기만 하고 다시 안 본다"
→ 주간 리뷰가 해결해줌. Dataview가 최근 7일 노트를 자동으로 보여주니까 훑어보기만 하면 됨.
→ 저녁 종료 때 "오늘 노트 회고"에 적으면 당일 리뷰도 됨.

### "백로그가 계속 쌓인다"
→ 주간 리뷰에서 10분만 투자. 안 할 거 삭제, 큰 거 쪼개기.
→ 마감일 없는 백로그는 "언젠가" 목록이니까 쌓여도 괜찮음. 스트레스 받지 말 것.

### "1시간 집중이 부담스럽다"
→ 15분만 해도 성공. 타이머 켜고 15분 후 "다음 행동 1줄"만 남기면 됨.
→ 핵심은 매일 뭔가 하나 건드리는 것.

### "급한 일이 생겼다"
→ 아무 노트에 `- [ ] 급한 일 📅 오늘날짜` 적기
→ 대시보드 "오늘/연체"에 바로 나타남

### "아이디어가 떠올랐는데 지금 정리할 시간이 없다"
→ `0-inbox/inbox.md`에 한 줄 던져두기
→ 주간 리뷰에서 정리 (적절한 노트로 옮기거나 삭제)
