	- API 캐싱으로 얼마 이득, brotli 번들 사이즈 압축으로 (이거 근데 옵션 그냥 킨거잖아?)

			- TPS 개선기 (제목부터 나의 경험에 대한 이야기를 하겠다로 어필해서 시선을 끌 수 있다.)
				- 1. API 캐싱, 2. next/image를 활용한 이미지 최적화(구체적으로 어떻게 최적화 작성)
				- 3. brotli 를 통한 번들 사이즈 압축(얼마나 압축) 
				- 각 과정마다 TPS가 얼마나 나아졌고 총 얼마나 개선이 했는지
