현재 Java Spring, Vue2 로 풀스택 개발을 하고 있는 4년차 개발자입니다. React, Next 를 사용하는 프론트엔드 개발자로 이직을 하고 싶은데 아무래도 실무 경험이 부족한 탓에 고민이 많은데 이를 해결하고싶습니다. 지금은 사이드 프로젝트로 포트폴리오를 만들어서 지원하고있는데 서류합격조차 안되고있는데 이부분도 함께 피드백 받고싶습니다. 감사합니다.

현재 4년차 웹 풀스택 개발자이고, 프론트엔드 개발자로 이직을 목표로 하고있어요.

회사에서는 주로 Vue를 쓰고있지만 개인적으론 React를 더 선호해서 React를 사용하는 회사를 원하고 사이드 프로젝트도 주로 React와 Next로만 하고있습니다. 

하지만 아무래도 React 실무 경험이 없다보니 사이드 프로젝트를 진행하면서도 이걸 경험이라고 할 수 있을까하는 의구심도 들고, 이걸로 포폴을 만들어서 지원해도 계속 서류탈락만하고 있어서 점점 더 확신이 없어지는것 같습니다. 

그래서 앞으로 어떻게 준비해야하는지에 대한 방향성과 이력서, 포트폴리오의 피드백을 부탁드리고 싶습니다.
