기술 역량 관련 고민

저는 디자이너 출신 프론트엔드 개발자로서, 퍼블리싱과 UI 구현 역량에는 강점이 있다고 생각합니다.

**실제로 팀 프로젝트에서는 메인 페이지 UI와 퍼포먼스 개선(React-Virtuoso, AVIF 변환, Lighthouse 점수 98점 등)을 담당했고, 디자인 경험을 살려 사용자 흐름 중심으로 기능을 구현해왔습니다.**

  

하지만 이력서에 이런 강점이 도드라지지 않게 느껴집니다. 특히 아래 부분이 고민됩니다:

  

**디자인 역량을 너무 강조하면 프론트엔드 지원자로 보이지 않을까?**

  

**퍼포먼스 최적화나 가상 리스트 구현 등 실질적인 기술 해결 경험이 있는데, 임팩트 있게 전달하지 못한 것 같아요.**

  

**프론트엔드 심화 기술(Next.js, React Query, Zustand 등)을 사용했지만, 성과나 문제 해결'중심으로 잘 녹여내지 못한 느낌입니다.**

  

**JAMMIT 프로젝트에서 백엔드 연동(Mux 업로드, 댓글, 대댓글, 좋아요, 조회수, 인증)까지 했지만 프론트엔드로서 강점을 흐릴까봐 애매하게 정리했습니다.**
