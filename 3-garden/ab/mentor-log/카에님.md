-   기술면접 준비에 도움을 받고 싶습니다. 기존에는 스타트업에서만 일을 했고, 이번에 이직 준비를 하면서 it 대기업 면접은 처음 보는 상황이라 걱정이 많이 됩니다. 스타트업과 기술면접 수준이나 내용이 아예 다를 것 같아서 도움이 필요합니다. 서류를 최근 넣기 시작한거라 그 이전 단계가 진행중이고, 면접 일정은 아직 잡히지 않았지만 멘토링 예약시점 정도가 되면 하나 정도는 잡힐 것 같습니다. 현재 서류통과되어 코딩테스트/과제전형 진행중인 곳은 라인플러스, 카카오뱅크, 토스플레이스, 여기어때, 오늘의집, 리디, 놀유니버스, 위메이드입니다. 이력서가 대부분은 같고 지원동기 정도만 살짝 다른데, 멘토링 이전에 필요하시다면 라인과 카카오뱅크의 이력서를 보내드리겠습니다! 감사합니다. 잘 부탁드리겠습니다.

	- CI/CD 구축 경험 있는지?
		- GitHub Actions를 사용하여 Next.js 기반 프로젝트의 CI/CD 파이프라인을 구축한 경험이 있습니다.  주요 단계는 Lint → Unit Test → Build → Docker 이미지 빌드 및 푸시 → Argo CD를 통한 GKE 자동 배포 순서로 구성했고, PR마다 코드 커버리지를 확인할 수 있도록 Codecov를 연동했습니다. gitops를 통해 배포이력, 상태를 git에서 일관되게 관리함.선언적으로 컨테이너 관리하기 떄문에 어떻게 가 아닌 어떤 상태가 되어야하는지를 명시하여 버전관리나 롤백이 용이하고 예측 가능성이 높다.(어떻게로 구현하면 어제는 되는데 오늘 안될수도;)
