## 1. 일반적인 사실에 대한 if

- 습관, 경험, 반복되는 사실 -> 조건절과 주절 둘 다 현재형

If he doesn't say anything, that means he's upset
If it smells bad, it's probably not safe to eat
If there are too many people, it's better to go somewhere else.
경험, 습성에 대해 알고 있다면 위와 같이 얘기할 수 있음.

> 아마: probably
> 다른데 가는게 더 좋아, 나아 (it's better to ~)

## 2. __현재의 사실__ 에 대한 가정 (현재 사실과 다른 가정)

- 실제 사실과는 다르게 상상해서 말할 때:__If+과거형, 주절에 could/would__

I live far from school 는 현재 사실. 그래서 어떻다와 같은 결론
I live far from school, so it takes a lot of time to go to school in the morning.

위와 같은 현재의 사실이 있다고 하자. 

- If I lived closer to school, I could save more time.
	- __can (X): could === would be able to__
		- 
	- __If I live (X)__
- If I ha d more money, I could go italy

I go to work 6 times a week, so it's hard to make plans on weekend.
- If I went to work 5 times a week, I could make plans on weekend.

I have an external hard drive, so I can carry my music and movies wherever I go. But If I didn't have one, I coudn't do that.

> one: 일반적인 대명사
> 

I gain weight quite easily so I have to be careful about what I eat. But If I didn't gain weight so easily, I wouldn't have to be careful about what I eat.

음식을 조심하지 않아도 __돼__ 가 아닌 __하지 않아도 됐을거야__ 와 같은 느낌을 줄 수 있다. 

- 현재 어떤 사실을 가지고 if 조건절로 구성을 하는거.
