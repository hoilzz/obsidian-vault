## 업적 평가
- (업적 평가) 나의 조직 성과에 어느 정도 기여 했는가?

- 타파스 메인 홈 개편과 스프린트 리딩을 맡아 일정 내에 개발 완료
- nGrinder를 활용해 타파스 서비스의 부하 테스트를 진행하고 병목 원인을 제거하여 안정성을 높임
- Figma 플러그인 `Icon Extractor`를 개발하여 디자인 리소스를 자동으로 추출하고, 개발 단계에서 별도 작업 없이 즉시 사용할 수 있도록 시스템화
- GCP 쿠버네티스를 활용한 AI 프로젝트 인프라 세팅
	- GCP k8s 세팅을 처음 사용했지만 시행착오를 겪으며 문제를 해결
	- 이 과정에서 얻은 해결 방법들을 문서화하여 공유
- pnpm과 yarn의 비교 문서를 작성하고 이를 기반으로 pnpm 도입
- Docker 빌드 최적화 작업을 통해 이미지 크기를 30% 줄이고, 빌드 시간을 7분에서 4분으로 단축하여 배포 효율성 향상
- Next.js 14 App Router 도입하여 성능과 개발자 경험 개선
- 타파스 메인홈 개편, AI 프로젝트의 리딩을 맡아 일정 내 개발 완료
