React의 컨커런트 렌더링은 우선순위가 높은 작업을 먼저 처리한다. as-is, to-be를 비교하면 된다.
실제 활용은 Suspense, Transtion으로 할 수 있다. 특히 useTransition이 사용자 인터페이스와 즉시 상호작용 한다.
_as-is_
```
무거운 렌더링 or 상태 로직 오래 걸릴 경우 사용자 입력을 무시한다.
```
_to-be_
```
사용자 입력이 발생하면 렌더링을 멈춘다.
입력 처리 완료 후 렌더링 작업을 다시 이어서 처리한다
```
