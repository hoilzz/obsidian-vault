만약에 노드 서버를 띄우는 이미지를 만든다고 해보자.

1. 이미지를 생성하기 위해 Dockerfile 파일 추가
2. Dockerfile에 다음과 같이 추가
```
FROM node:latest

# 패키지 설치를 하기 위해 복붙
COPY ./packages* /usr/src/app/
WORKDIR /usr/src/app
RUN npm install

COPY . /usr/src/app

EXPOSE 8080
CMD node app.js
```
