	- 404 (`접근할 수 없는 페이지, 서비스 이용에 불편을 드려 죄송합니다. 요청하신 페이지에 접근할 수 없습니다.`)

- API 호출 실패시 : `본인/보호자 인증에 실패했습니다. 확인 후 다시 시도해주세요.`

	- setAgreement (인증 설정)

- 캐시가 정리해준 result_code 별 메시지
