## 웹폰트 로드 전까지 렌더링 블락
- CSSOM을 생성하는 과정에서 발생
- 웹 폰트 링크로 정의된 코드를 만나고 해당 폰트 파일 다운로드 시작
- paint 단계에서 다운로드 완료 전까지 렌더링 차단
- 요 단계에서 웹폰트 최적화가 필요함

### 최적화 1  - 폰트 파일 용량 줄이기

##### woff2 사용하기(woff 보다 압츅률 30~50%높음)
	- ie제외하고 전부 사용 가능
- 용량이 작은 폰트를 먼저 적용하여 웹폰트 최적화한 @font-face 규칙을 보자
```css
@font-face {
	font-family: ...
	src: url($$$.woff2) format(woff2),
	url(@@@.woff) format(woff)
}
```
