### 채널톡 방식


1. Figma의 선택한 프레임 내부의 컴포넌트를 모두 가져옵니다
   -> 이거 수동으로 플러그인 실행 해줘야함. 그냥 아이콘 추가될때마다(hook) 알아서 실행하는 거 없을까?
2. 해당 컴포넌트의 id들을 파라미터로 figma API를 호출하여 svg 파일을 가져옵니다
3. svg 파일을 Git Blob으로 변환합니다
4. config에 명시된 레포지토리 정보로 타겟 브랜치의 Git Reference를 가져옵니다
5. 해당 Ref의 sha값으로 Git Commit을 가져옵니다
6. 해당 Commit의 sha값으로 Root Git Tree를 가져옵니다
7. Root부터 입력한 추출 경로에 해당하는 Git Tree를 찾습니다
8. 찾으면, 해당 Git Tree의 File을 3번에서 변환한 Git Blob으로 갈아 끼웁니다. 이걸 새로운 Git Tree로 만듭니다
9. 변경을 인식하게 하려면, 루트부터 바뀐 경로까지 모든 Git Tree의 sha값이 새롭게 변경되어야합니다. 역순으로 7번에서 탐색했었던 트리가 8번 Tree의 sha를 참조하도록, 이를 통해 만들어진 트리의 sha를 상위 트리가 참조하도록 반복해서 루트 트리까지 새로 생성합니다
10. 5번의 Commit을 부모로 하는, 새로운 루트 트리를 내용으로 하는 Git Commit을 생성합니다
11. 10번의 Commit을 베이스로 하는 Git Reference(Git branch)를 생성합니다
12. 11번의 브랜치에서 config에 명시한 브랜치로 향하는 PR을 생성합니다
