### 정의
Retrieval-Augmented Generation
__LLM과 결합하여 외부 정보를 실시간으로 검색하는 시스템__
RAG는 LLM이 데이터를 생성하는 것에 더해, 필요한 정보를 외부에서 찾아서 답변을 더 정확하게 하는 것이다. 즉, 모델이 답변을 생성하기 전에 외부 DB나 문서에서 관련 정보를 실시간으로 검색하여 생성된 텍스트를 만든다.
### 배경
기존 LLM은 학습된 데이터만을 기반으로 작동한다. 즉 LLM은 __기억에 있는 정보__ 만을 활용하며, 즉각적인 데이터 검색은 불가하다.

### 작동 방식

1. 사용자가 질문 입력
2. RAG 모델은 외부 DB나 문서 저장소에서 관련 정보를 검색한다.
3. 검색된 정보를 바탕으로 답변을 생성
