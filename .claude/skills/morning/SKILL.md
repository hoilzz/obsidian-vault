# 아침 루틴 (3분)

오늘 날짜 기준으로 개인 아침 루틴을 진행한다.

## 순서

1. **오늘 데일리 노트 확인**: `journal/0-d/` 에서 오늘 날짜(YY-MM-DD 형식) 파일을 읽는다. 없으면 사용자에게 알린다.

2. **미완료 태스크 수집**: 볼트 전체에서 `- [ ]` 태스크를 검색한다.
   - `4-worklog/`, `template/`, `.obsidian/`, `.kiro/`, `journal/0-d/` 경로는 제외
   - 마감일(📅) 있는 것 중 오늘 이전인 것(연체) 우선 표시
   - `#focus/*` 태그 붙은 것 별도 표시
   - 나머지 백로그도 간단히 보여주기

3. **사용자에게 선택 요청**: "오늘의 1시간에 뭘 할까요?" 라고 물어본다.
   - 후보 목록을 보여주고 번호로 고르거나 새로 말할 수 있게 한다

4. **데일리 노트 업데이트**: 사용자가 선택하면 데일리 노트의 `오늘의 1시간(딱 하나)` 섹션에 태스크를 작성한다.
   - 잡일도 추가하고 싶은지 물어본다

5. **마무리**: 오늘 계획을 한 줄로 요약해서 보여준다.

## 규칙
- 한국어로 대화
- 간결하게, 3분 안에 끝낼 수 있도록
- 데일리 노트에 쓸 때 기존 템플릿 구조를 유지할 것
