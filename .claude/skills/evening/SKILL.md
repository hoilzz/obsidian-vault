# 저녁 루틴 (2분)

오늘 날짜 기준으로 개인 저녁 루틴을 진행한다.

## 순서

1. **오늘 데일리 노트 읽기**: `journal/0-d/` 에서 오늘 날짜(YY-MM-DD 형식) 파일을 읽는다.

2. **오늘 한 일 확인**: 데일리 노트의 태스크 완료 여부를 확인하고, 오늘 수정된 노트 목록을 간단히 보여준다.
   - `git diff --name-only` 또는 파일 수정시간 기준으로 오늘 변경된 파일 확인
   - `4-worklog/`, `.obsidian/`, `template/` 제외

3. **종료 섹션 대화**: 사용자에게 순서대로 물어본다:
   - "오늘 한 줄 기록 — 뭐 했어요?"
   - "내일의 1개 — 내일 뭐 할까요?"
   - 오늘 수정한 노트 목록을 보여주며 "노트 회고 — 추가할 메모 있어요?"

4. **데일리 노트 업데이트**: 답변을 종료 섹션에 작성한다.
   - `오늘 한 줄 기록:` 뒤에 작성
   - `내일의 1개:` 뒤에 작성
   - `오늘 만든/수정한 노트:` 뒤에 작성

5. **미완료 태스크 처리**: 오늘 데일리 노트에 미완료 태스크가 있으면 어떻게 할지 물어본다:
   - 완료 처리 / 내일로 이월(📅) / 백로그로(마감 제거)

## 규칙
- 한국어로 대화
- 간결하게, 2분 안에 끝낼 수 있도록
- 데일리 노트에 쓸 때 기존 템플릿 구조를 유지할 것
