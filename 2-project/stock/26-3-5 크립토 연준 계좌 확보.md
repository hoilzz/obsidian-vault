연준의 Limited purpose account 확보
- 연준 결제 인프라 접근권 (제휴은행 없음)
- 은행들 대규모 반발(밥그릇 싸움)


### 연준 마스터 계정

- 미국 금융 시스템에서 가장 중요.
- 연준에 직접 계좌를 가진 기관만 미국 금융 시스템의 코어에 들어갈 수 있음
	- 보통은 은행 (등) 만 가질 수 있음

### 이 계좌를 확보하면 뭐가 좋은가?

- 연준 지급결제 시스템 사용
- 은행간 결제
- 준비금 관리
- FedWire 사용


### FedWIre?

미국의 핵심 결제망

- 실시간 결제
- 거래취소 불가
- 수조 달러 매일 이동

- 사용 예
	- 은행 간 송금
	- 국채 결제
	- 대기업 대규모 송금

```
JP Morgan -> Citi // 1000억 송금
```

###  크립토 업계에서 중요한 사건인 이유

기존에는
- 크립토 거래소 -> 중개은행 -> Fed
	- ex. 업비트가 케이뱅크 끼고 하는것

현재
- Kraken -> FedWire 직접 접속

Fedwire 접근 가능하다면
	- USD <-> crypto 전환 빠름
	- 즉, USDC 즉시 발행
	- 기존에는 당연히 은행 처리 -> 다음날 -> USDC 발행

즉, 중개은행이 사라진다.
- 비용 감소, 은행 리스크 감소

### 중요한 제한점

- 기존은행이 누리는 중앙은행 서비스(지급준비금 이자)는 미포함
- Fedwire 결제망은 사용가능하지만 완전한 은행은 아니다.
	- 지급준비금 이자 X, 은행 대출 창구 X
